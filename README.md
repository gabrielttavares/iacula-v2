# Iacula-v2

## tech

- Tauri v1
- TypeScript
- Rust
- HTML/CSS
- Docker

## install

1. Clone:
```bash
git clone https://github.com/seu-usuario/iacula-v2.git
cd iacula
```

2. Install Node dependencies:
```bash
npm install
```

3. Install Rust dependencies:
```bash
cd src-tauri
cargo build
```

## execute the app

dev env:
```bash
npm run dev
```

prod build:
```bash
npm run build
```
